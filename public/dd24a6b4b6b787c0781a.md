## 安全に容量を空ける方法

### 1. MacBookのキャッシュを削除する

ターミナルを開いて

```
rm -rf ~/Library/Caches/*
```

再起動すれば、必要な分は自動で再生成されます。

### 2. ブラウザのキャッシュを削除

Chrome / Edge / Firefox などは、
使い続けると数GB以上のキャッシュが溜まることがあります。

``３点リーダー→ブラウジングデータ→データを削除``



### 3. Docker のビルドキャッシュ削除
```
docker builder prune -a
```
使われていないビルドキャッシュを全削除

数GB〜数十GB空くこともある

### 4. Node.js のキャッシュを削除
```
npm cache clean 
```
また、プロジェクト内の
### 5. 不要なアプリを削除

アプリケーションフォルダをサイズ順に並び替えて、
もう使っていないアプリを削除します。

特に容量を使うことが多いのは

* Adobe系
* Xcode
* などなど

### 6.ゴミ箱を空にする
定期的に消さないと溜まっていくのでここも確認しとくといいと思います！

### 7. 再起動する

再起動だけで削除される一時ファイルもあります。

長時間スリープ運用している場合は、
数週間に一度は再起動するのがおすすめです。

## まとめ
