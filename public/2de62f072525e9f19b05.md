## はじめに
自分の開発環境で使用している構成を、できるだけシンプルにまとめたものです。
余計なツールをあまり入れずに、実際の開発で必要だったものだけを厳選しています。
## 1.Homebrew
##### macに何をインストールするにもこのアプリが必要になります。
##### インストール方法
```bin
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
#### 詳細↓
https://brew.sh/ja/

## 2.おすすめのCLIアプリ
プログラミングするときに必要なものをまとめてみました
```
brew install git
brew install wget
brew install curl
brew install ffmpeg
brew install make
brew install cmake
brew install python
brew install node
brew install go
brew install ruby
brew install git-lfs
```
